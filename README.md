The User Management API is a backend system built using ASP.NET Core that provides a structured and secure way to manage user data. It supports full CRUD operations and includes features like role management, account activation status, and timestamp tracking.

The application follows industry best practices such as:

Use of DTOs for clean data transfer

Secure password storage using hashing

Separation of concerns for maintainable architecture

RESTful API design for easy integration with frontend applications

This project demonstrates strong backend development skills and can be extended with authentication (JWT), database integration, and cloud deployment.
